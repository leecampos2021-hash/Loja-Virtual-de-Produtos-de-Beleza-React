# Loja-Virtual-de-Produtos-de-Beleza-React
Este projeto é uma aplicação front-end desenvolvida em React que simula uma loja virtual de produtos de beleza. A proposta é praticar conceitos fundamentais de desenvolvimento web moderno, como componentização, renderização dinâmica de listas, filtros por categoria e organização de dados em JavaScript.

A interface foi pensada com uma estética clean e foco na experiência do usuário, permitindo navegar pelos produtos, buscar itens e filtrar por categorias como maquiagem, skincare, cabelo e perfume.
⚠️ Este projeto foi desenvolvido a partir de um código base fornecido pelo professor, com a proposta de que os alunos dessem continuidade e implementassem as funcionalidades solicitadas. Toda a lógica de listagem, filtros, organização dos dados e ajustes de estrutura foram implementados e adaptados por mim a partir desse ponto inicial.

Funcionalidades
  Listagem dinâmica de produtos a partir de um array de dados
  Filtro por categorias
  Campo de busca por nome do produto
  Cards de produto com imagem, marca, categoria e preço
  Layout em grid responsivo
  Estilização com CSS
  
Tecnologias utilizadas
  React
  JavaScript (ES6+)
  CSS

Estrutura de componentes reutilizáveis
 
 Organização
  products.js com os dados dos produtos
  Componentes para menu de categorias, barra de busca e cards
  Imagens servidas pela pasta public/images
 
Objetivo do projeto
  Este projeto foi desenvolvido como prática acadêmica para consolidar conhecimentos em:
  Manipulação de arrays com map e filter
  Passagem de props entre componentes
  Estruturação de uma aplicação React
