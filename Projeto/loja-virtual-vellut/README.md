# Catálogo de Produtos

Este projeto é um catálogo de produtos feito com React e Vite.

## Instalação

1. Abra o terminal na pasta do projeto.
2. Instale as dependências:

```bash
npm install
```

3. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

4. Abra o endereço mostrado pelo terminal (normalmente `http://localhost:5173`).

## Como fazer fork do projeto

Para criar uma cópia do repositório no seu próprio GitHub, siga estes passos:

1. Acesse o repositório original no GitHub.
2. Clique em **Fork** no canto superior direito.
3. Abra o repositório criado no seu perfil.
4. Copie o link de clone (HTTPS ou SSH).
5. No terminal, clone o seu fork:

```bash
git clone https://github.com/seu-usuario/catalogo-produtos.git
```

6. Entre na pasta do projeto:

```bash
cd catalogo-produtos
```

7. Instale as dependências e execute o projeto:

```bash
npm install
npm run dev
```

## Atualizar o fork com o repositório original

Se você quiser manter o seu fork atualizado, adicione o remoto upstream e traga as mudanças:

```bash
git remote add upstream https://github.com/usuario-original/catalogo-produtos.git
git fetch upstream
git merge upstream/main
```

> Substitua `usuario-original` pelo nome de usuário do GitHub do repositório original.
